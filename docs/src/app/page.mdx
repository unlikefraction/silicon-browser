# silicon-browser

**The most reliable browser for your AI agent.**

Stealth-first headless browser CLI that uses [CloakBrowser](https://cloakbrowser.dev/)'s patched Chromium (33 C++ patches) + 18 JavaScript evasions. Passes Cloudflare, DataDome, BrowserScan. Zero configuration.

```bash
npm install -g silicon-browser
silicon-browser install
silicon-browser open https://example.com
```

## Stealth Architecture

Four layers of anti-detection, zero configuration:

### Layer 1: CloakBrowser Binary
33 C++ source-level patches compiled into Chromium. TLS fingerprints, WebGL, Canvas, Audio — all handled at the engine level. Undetectable because there's nothing to detect.

### Layer 2: Chrome Launch Flags
`--disable-blink-features=AutomationControlled`, `--fingerprint=<seed>`, WebRTC leak prevention, GPU rasterization.

### Layer 3: JavaScript Injection
Defense-in-depth: `window.chrome` object, Permissions API, CDP artifact removal, `navigator.connection`.

### Layer 4: Network Headers
Realistic `Sec-Fetch-*`, `Accept`, `Accept-Language` headers matching real Chrome.

## Profiles

Every profile has its own cookies, history, and **pinned fingerprint**. Same profile = same identity.

```bash
# Default profile (always there)
silicon-browser open https://example.com

# Named profiles
silicon-browser --profile work open https://shopify.com
silicon-browser --profile personal open https://github.com

# Incognito (throwaway)
silicon-browser --incognito open https://example.com
```

## Push/Clone: Sync Profiles Anywhere

## Ref-Based Interaction

Instead of fragile CSS selectors, every element gets a ref like `@e1`, `@e2`:

```bash
silicon-browser open https://example.com
silicon-browser snapshot -i
# @e1  link "Home"
# @e2  button "Sign In"
# @e3  input "Search..."

silicon-browser click @e2
silicon-browser fill @e3 "query"
```

93% fewer tokens than screenshot-based approaches.

## All Commands

| Command | Description |
|---------|-------------|
| `open <url>` | Navigate to URL |
| `snapshot -i` | Get interactive elements with refs |
| `click @ref` | Click element |
| `fill @ref "text"` | Fill input field |
| `type "text"` | Type text |
| `screenshot` | Capture screenshot |
| `eval "js"` | Execute JavaScript |
| `scroll down` | Scroll the page |
| `tabs` | List open tabs |
| `push <name>` | Share profile over HTTP |
| `clone <url>` | Clone profile from push server |
| `profile list` | List all profiles |
| `close` | Close browser |
| `install` | Download CloakBrowser + Chrome |
