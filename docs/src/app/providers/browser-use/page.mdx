# Browser Use

[Browser Use](https://browser-use.com) provides cloud browser infrastructure for AI agents. Use it when running silicon-browser in environments where a local browser isn't available (serverless, CI/CD, etc.).

## Setup

```bash
export BROWSER_USE_API_KEY="your-api-key"
silicon-browser -p browseruse open https://example.com
```

Or use environment variables for CI/scripts:

```bash
export SILICON_BROWSER_PROVIDER=browseruse
export BROWSER_USE_API_KEY="your-api-key"
silicon-browser open https://example.com
```

The `-p` flag takes precedence over `SILICON_BROWSER_PROVIDER`.

When enabled, silicon-browser connects to a Browser Use cloud session instead of launching a local browser. All commands work identically.

Get your API key from the [Browser Use Cloud Dashboard](https://cloud.browser-use.com/settings?tab=api-keys). Free credits are available to get started, with pay-as-you-go pricing after.
