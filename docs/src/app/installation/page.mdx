# Installation

## Global installation (recommended)

Installs the native Rust binary for maximum performance:

```bash
pnpm add -g silicon-browser
silicon-browser install  # Download CloakBrowser + Chrome for Testing (first time)
```

This is the fastest option -- commands run through the native Rust CLI directly with sub-millisecond parsing overhead.

## Quick start (no install)

```bash
pnpm dlx silicon-browser install   # Download bundled browsers (first time only)
pnpm dlx silicon-browser open example.com
```

## Project installation (local dependency)

For projects that want to pin the version in `package.json`:

```bash
pnpm add silicon-browser
pnpm dlx silicon-browser install  # Download bundled browsers (first time)
```

Then use via `pnpm dlx` or `package.json` scripts.

## Homebrew (macOS)

```bash
brew install silicon-browser
silicon-browser install  # Download bundled browsers (first time)
```

## Cargo (Rust)

```bash
cargo install silicon-browser
silicon-browser install  # Download bundled browsers (first time)
```

Compiles from source (~2-3 min). Requires the Rust toolchain ([rustup.rs](https://rustup.rs)).

## From source

```bash
git clone https://github.com/unlikefraction/silicon-browser
cd silicon-browser
pnpm install
pnpm build
pnpm build:native
./bin/silicon-browser install
pnpm link --global
```

## Linux dependencies

On Linux, install system dependencies:

```bash
silicon-browser install --with-deps
```

## Updating

Upgrade to the latest version:

```bash
silicon-browser upgrade
```

Detects your installation method (npm, Homebrew, or Cargo) and runs the appropriate update command automatically. Displays the version change on success, or informs you if you are already on the latest version.

## Custom browser

Use a custom browser executable instead of bundled Chromium:

- **Serverless** - Use `@sparticuz/chromium` (~50MB vs ~684MB)
- **System browser** - Use existing Chrome installation
- **Custom builds** - Use modified browser builds

```bash
# Via flag
silicon-browser --executable-path /path/to/chromium open example.com

# Via environment variable
SILICON_BROWSER_EXECUTABLE_PATH=/path/to/chromium silicon-browser open example.com
```

### Serverless example

Use `@sparticuz/chromium` or similar to obtain a Chromium executable path, then pass it via `--executable-path` or `SILICON_BROWSER_EXECUTABLE_PATH`.

## AI agent setup

silicon-browser works with any AI agent out of the box. For richer context:

### AI coding assistants (recommended)

Install the skill for your AI coding assistant:

```bash
pnpm dlx skills add unlikefraction/silicon-browser
```

This works with Claude Code, Codex, Cursor, Gemini CLI, GitHub Copilot, Goose, OpenCode, and Windsurf. The skill is fetched from the repository and stays up to date automatically.

> **Do not** copy `SKILL.md` from `node_modules` -- it will become stale as new features are added. Always use `pnpm dlx skills add` or reference the repository version.

### AGENTS.md / CLAUDE.md

Add to your instructions file:

```markdown
## Browser Automation

Use `silicon-browser` for web automation. Run `silicon-browser --help` for all commands.

Core workflow:
1. `silicon-browser open <url>` - Navigate to page
2. `silicon-browser snapshot -i` - Get interactive elements with refs (@e1, @e2)
3. `silicon-browser click @e1` / `fill @e2 "text"` - Interact using refs
4. Re-snapshot after page changes
```
